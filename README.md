## Instalar

### Instalar las dependencias y activar el virtual env
utilizar una terminal en bash

```bash
pip install poetry

poetry install
poetry shell
```

### Aplicar Migraciones

```bash
python source/manage.py migrate
```

### Para iniciarlo
Utilizar el comando

```bash
python source/manage.py runserver
```
